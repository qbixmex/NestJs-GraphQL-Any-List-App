## Description

This is Tutorial for learning NestJS with GraphQL

## Installation

```bash
# NPM
$ npm install

#YARN
$ yarn
```

## Environment Variables

__Rename ```.env.template``` to ```.env``` and put your local credentials__

## Running the app

```bash
# development
$ npm run start # NPM
$ yarn start # YARN

# watch mode
$ npm run start:dev # NPM
$ yarn start:dev # YARN

# production mode
$ npm run start:prod # NPM
$ yarn start:prod # YARN
```

## Test

```bash
# unit tests
$ npm run test # NPM
$ yarn test # YARN

# e2e tests
$ npm run test:e2e # NPM
$ yarn test:e2e # YARN

# test coverage
$ npm run test:cov # NPM
$ yarn test:cov # YARN
```
